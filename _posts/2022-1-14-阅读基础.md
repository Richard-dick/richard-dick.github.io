# 阅读基础

> 雅思阅读共三篇文章，40道题，包括多选、完成句子、完成图标、选择标题、判断对错、分类和匹配选项等题型。总共时间限定在一小时内。本文依托于胡敏雅思教材第七代-雅思阅读基础版的12个单元，记录其中重要的知识点。工具书，应该越读越薄。

## Chapter-1-Friendship

1. spend times with friends
   1. --informally--> **hang out with them**
   2. --formally--> **socialize with them**

2. karate n.空手道
   - Being a combat sport, karate carries with it the risk of injury.

3. caravan n.大篷车 v.乘大篷车旅行
   - Old-fashioned Gypsy caravans are painted wooden vehicles that are pulled by horses.
  
4. go多用于-ing式单词
    - go swimming // go dancing // go shopping // go skiing

5. play多用于竞争性较强的活动
    - play chess // play board game // play poker 

6. do则是一些娱乐性，强身的活动
   - do karate // do exercise // do kick boxing // do weightlifting // do puzzles

7. reader 无论何时都不能作为选择的依据

8. 不是所有人都会上网，许多限定词会以这样的形式出现。所以不能说所有人都有多少多少网友。

## Chapter-2-Body and Mind

1. club所指的球棒多是一头粗，一头细，如棒球棒或者高尔夫球棒。而bat则多为棒球，板球，乒乓球或者羽毛球。

2. 一些使用板子board的运动
   - ski // surfing // windsurfing

3. 回答问题时，会要求词数限制，且不能使用如it's式的缩写。
